Projects and Research

Asset Allocation via Machine Learning and Factor Models
Bachelor Thesis, USI · 04/2026 · Supervisor: Prof. Patrick Gagliardini

An empirical evaluation of portfolio optimization approaches. Python backtesting framework for multi-asset strategies, integrating Fama-French models and machine learning (Neural Networks, Elastic Net) for return forecasting, with Mean-Variance Optimization under Ledoit-Wolf shrinkage and turnover constraints.

A Quantitative Approach to Commodity Diversification
Working Paper, Starting Finance Club USI · Ongoing · with Gianluca Ladelfa

Regime-dependent correlations and algorithmic optimization. Python framework analyzing regime-dependent correlations between commodities and the S&P 500, with Mean-Variance optimization (SLSQP) to construct efficient frontiers, evaluated against CFTC COT data.

Constructing Robust Investment Strategies
BSE Technical Project · 07/2025 · Supervisor: Prof. Javier Gil-Bazo

Regularization techniques and CVaR optimization. Python framework for robust portfolio construction using LASSO/Ridge models, solving VaR-constrained portfolios via SciPy's SLSQP with 5-fold cross-validation, and a dual risk framework (Parametric vs. Historical Bootstrapping).
