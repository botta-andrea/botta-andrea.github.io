Asset Allocation via Machine Learning and Factor Models: An Empirical Evaluation of Portfolio Optimization Approaches
Bachelor Thesis, USI
May 2026
Supervisor: Prof. Patrick Gagliardini (USI, Swiss Finance Institute)

Python backtesting framework for multi-asset strategies, integrating Fama-French factors and Goyal et al. predictors via machine learning (Neural Networks, Elastic Net) for return forecasting, with Mean-Variance Optimization under Ledoit-Wolf shrinkage and turnover constraints.
